# ImageLab

A university project for the **Object-Oriented Programming** course in C++.

ImageLab is a console-based C++ application for loading and processing images in the NetPBM family of formats (.ppm, .pgm, .pbm). It allows users to build custom filter pipelines per image and apply them individually or all at once.

**Supported filters:**
- blur
- sharpen
- negative
- sobel
- threshold
- normalize / contrast-stretching
